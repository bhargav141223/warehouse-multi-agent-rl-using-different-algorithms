================================================================================
QUESTION 1: What is the project problem being solved?
================================================================================

ANSWER:
The project solves the Multi-Agent Warehouse Navigation Problem using 
Reinforcement Learning. 

PROBLEM DESCRIPTION:
In a warehouse setting, multiple autonomous agents (robots) need to navigate 
from their starting positions to a goal location (delivery point) while:

1. Avoiding collisions with each other (multi-agent coordination)
2. Avoiding static obstacles (walls, shelves, narrow corridors)
3. Avoiding dynamic obstacles (moving objects, other robots)
4. Reaching the goal as quickly as possible (path efficiency)
5. Learning cooperative behaviors (not blocking each other)

CHALLENGES ADDRESSED:
- Sparse Rewards: Traditional RL only gives reward at goal (+10), making 
  learning slow. Solution: LLM-based dense reward shaping (+2 for progress)
  
- Sample Inefficiency: Agents need millions of trials. Solution: RAG memory 
  retrieves similar past experiences to accelerate learning
  
- Credit Assignment: Hard to know which agent caused collision. Solution: 
  Individual agent policies with shared LLM feedback
  
- Coordination: Multiple agents may compete for same path. Solution: Spread 
  bonus (+0.3) in LLM rewards encourages exploration

ENVIRONMENT COMPLEXITY LEVELS:
- Simple: 5×5 grid, 2 agents, no obstacles (basic learning)
- Medium: 8×8 grid, 3 agents, static obstacles (navigation)
- Complex: 10×10 grid, 4 agents, narrow corridors (pathfinding)
- Dynamic: 8×8 grid, 4 agents, moving obstacles (adaptation)

ANSRAMETER 1: EPISODE REWARD (Cumulative Reward)
------------------------------------------------------------------------
DEFINITION: Sum of all rewards (environment + LLM) received by all agents 
in a single episode.

MATHEMATICAL FORMULA:
    T      num_agents
R = Σ      Σ      [R_env(i,t) + λ × R_LLM(i,t)]
   t=1      i=1

Where:
- T = total steps in episode
- R_env = environment reward (-0.1 step, -5 collision, +10 goal)
- R_LLM = LLM shaped reward (+2 progress, -2 collision, +0.3 spread)
- λ = 0.5 (weighting factor)

INTERPRETATION:
- Positive value (>0): Good performance, goal reached with few penalties
- Negative value (<0): Poor performance, collisions or timeout occurred
- Range typically: -50 to +30

GOOD VALUES:
- Excellent: +20 to +30 (quick goal, no collisions)
- Good: +10 to +20 (goal reached, minor penalties)
- Poor: -50 to -10 (multiple collisions or timeout)

CODE FOR CALCULATION:
```python
# In train.py - Episode reward calculation
ep_reward = 0.0
for step in episode:
    env_rewards = [-0.1] * num_agents  # Step penalty
    if collision:
        env_rewards[i] -= 5.0  # Collision penalty
    if goal_reached:
        env_rewards[i] += 10.0  # Success reward
    
    llm_rewards = llm_feedback(...)  # Get LLM rewards
    
    # Combined reward
    combined_rewards = [
        env_rewards[i] + 0.5 * llm_rewards[i] 
        for i in range(num_agents)
    ]
    
    ep_reward += sum(combined_rewards)
```

PLOT TYPE: Line Chart
- X-axis: Episode number
- Y-axis: Total reward
- Shows learning curve over time

PARAMETER 6: VALUE FUNCTION ESTIMATE (Critic Accuracy)
------------------------------------------------------------------------------
DEFINITION: How well the Critic network predicts actual returns. Measures 
the accuracy of value estimation.

MATHEMATICAL FORMULA:
Value_Error = E[ |V(s_t) - G_t| ]

Where:
- V(s_t) = Critic's predicted value at state s_t
- G_t = Actual discounted return from state s_t

INTERPRETATION:
- Low error (<1.0): Critic accurately predicts future rewards
- High error (>5.0): Critic poorly estimates, may destabilize training
- Should decrease over training

CODE FOR CALCULATION:
```python
# In agent.py - during MAPPO update
values_t = torch.stack(self.values).detach()
returns_t = torch.FloatTensor(returns).unsqueeze(1)

# Value function loss (MSE)
critic_loss = nn.MSELoss()(new_values, returns_t)
value_error = critic_loss.item()
```

------------------------------------------------------------------------------
PARAMETER 7: POLICY ENTROPY (Exploration)
------------------------------------------------------------------------------
DEFINITION: Measure of randomness in the agent's action selection. Higher 
entropy = more exploration.

MATHEMATICAL FORMULA:
H(π) = - Σ_a π(a|s) × log(π(a|s))

Where:
- π(a|s) = probability of action a in state s
- Sum over all 5 possible actions

INTERPRETATION:
- High entropy (~1.6 for 5 actions): Agent exploring randomly
- Medium entropy (~0.8): Balanced exploration/exploitation
- Low entropy (~0.1): Agent confident, mostly deterministic
- Should decrease slowly over time (from exploration to exploitation)

CODE FOR CALCULATION:
```python
# In agent.py - MAPPO update
probs = self.actor(states_t)
dist = Categorical(probs)
entropy = dist.entropy().mean().item()

# Entropy bonus in loss function
loss = actor_loss + 0.5 * critic_loss + 0.01 * (-entropy)
```

PLOT TYPE: Line Chart
- X-axis: Episode
- Y-axis: Average entropy
- Should show gradual decrease

------------------------------------------------------------------------------
PARAMETER 8: MAPPO CLIPPING RATIO
------------------------------------------------------------------------------
DEFINITION: Measures how much the policy is being constrained by MAPPO clipping.

MATHEMATICAL FORMULA:
r_t(θ) = π_θ(a_t|s_t) / π_θold(a_t|s_t)

INTERPRETATION:
- Ratio near 1.0: Policy not changing much (stable)
- Ratio > 1.2 or < 0.8: Large policy updates (may be unstable)
- Clipping kicks in when |r - 1| > 0.2 (epsilon)

CODE FOR CALCULATION:
```python
# In agent.py
old_log_probs_t = torch.stack(self.log_probs).detach()
new_log_probs = dist.log_prob(actions_t)

# Probability ratio
ratio = torch.exp(new_log_probs - old_log_probs_t)
clip_fraction = (abs(ratio - 1) > clip_eps).float().mean().item()
```

PARAMETER 12: AGENT COORDINATION METRICS
------------------------------------------------------------------------------
A. Average Pairwise Distance (Spread)
DEFINITION: How spread out agents are from each other.

FORMULA:
spread = (1/N_pairs) × Σ_{i<j} Manhattan(agent_i, agent_j)

B. Path Intersection Count
DEFINITION: Number of times agents cross the same cell (potential collision risk)

C. Synchronization Score
DEFINITION: How often agents move in coordinated directions

CODE FOR CALCULATION:
```python
# Spread calculation
def compute_spread(positions):
    if len(positions) < 2:
        return 0.0
    total = 0.0
    count = 0
    for i in range(len(positions)):
        for j in range(i + 1, len(positions)):
            total += abs(positions[i][0] - positions[j][0]) + \
                     abs(positions[i][1] - positions[j][1])
            count += 1
    return total / count if count > 0 else 0.0
```

import matplotlib.pyplot as plt
import numpy as np

def plot_reward_curve(rewards_per_episode, window=20):
    """
    Plot episode rewards with moving average smoothing.
    
    Args:
        rewards_per_episode: List of total rewards for each episode
        window: Window size for moving average
    """
    fig, ax = plt.subplots(figsize=(10, 6))
    
    episodes = range(1, len(rewards_per_episode) + 1)
    
    # Raw rewards
    ax.plot(episodes, rewards_per_episode, 'b-', alpha=0.3, label='Raw Reward')
    
    # Moving average
    if len(rewards_per_episode) >= window:
        moving_avg = np.convolve(rewards_per_episode, 
                                  np.ones(window)/window, mode='valid')
        ma_episodes = range(window, len(rewards_per_episode) + 1)
        ax.plot(ma_episodes, moving_avg, 'b-', linewidth=2, 
                label=f'{window}-Episode Moving Avg')
    
    ax.axhline(y=0, color='r', linestyle='--', alpha=0.5, label='Zero Line')
    ax.set_xlabel('Episode')
    ax.set_ylabel('Total Reward')
    ax.set_title('Learning Curve: Episode Rewards')
    ax.legend()
    ax.grid(True, alpha=0.3)
    
    # Add trend annotation
    if len(rewards_per_episode) > 50:
        early_mean = np.mean(rewards_per_episode[:25])
        late_mean = np.mean(rewards_per_episode[-25:])
        improvement = late_mean - early_mean
        ax.text(0.02, 0.98, f'Improvement: {improvement:.2f}', 
                transform=ax.transAxes, verticalalignment='top',
                bbox=dict(boxstyle='round', facecolor='wheat', alpha=0.5))
    
    plt.tight_layout()
    return fig

class RLEvaluator:
    """
    Comprehensive evaluator for Warehouse Multi-Agent RL System.
    """
    
    def __init__(self, env_name="Simple", num_eval_episodes=50):
        self.env_name = env_name
        self.num_eval_episodes = num_eval_episodes
        self.env = None
        self.agents = []
        self.results = {
            "rewards": [],
            "lengths": [],
            "successes": [],
            "collisions": [],
            "entropy": [],
            "value_estimates": [],
            "agent_positions_history": []
        }
    
    def setup(self):
        """Initialize environment and agents."""
        ENV_MAP = {
            "Simple": SimpleEnv,
            "Medium": MediumEnv,
            "Complex": ComplexEnv,
            "Dynamic": DynamicEnv
        }
        
        self.env = ENV_MAP[self.env_name]()
        obs = self.env.reset()
        obs_dim = len(obs[0])
        
        # Load trained agents (or create new for testing)
        self.agents = [MAPPOAgent(obs_dim=obs_dim, action_dim=5) 
                       for _ in range(self.env.num_agents)]
        
        print(f"Evaluation setup complete for {self.env_name} environment")
        print(f"  Grid size: {self.env.grid_size}×{self.env.grid_size}")
        print(f"  Agents: {self.env.num_agents}")
        print(f"  Evaluation episodes: {self.num_eval_episodes}")
    
    def run_evaluation(self):
        """Run evaluation episodes and collect metrics."""
        print("\nStarting evaluation...")
        
        for ep in range(self.num_eval_episodes):
            obs = self.env.reset()
            episode_reward = 0.0
            episode_collisions = 0
            episode_entropy = []
            episode_values = []
            positions_history = [list(self.env.agent_positions)]
            step_count = 0
            done = False
            
            while not done and step_count < 200:
                actions = []
                log_probs = []
                values = []
                
                # Each agent selects action
                for i, agent in enumerate(self.agents):
                    action, log_prob, value = agent.select_action(obs[i])
                    actions.append(action)
                    log_probs.append(log_prob)
                    values.append(value)
                    
                    # Track value estimates
                    episode_values.append(value.item())
                
                # Calculate entropy from action probabilities
                import torch
                obs_t = torch.FloatTensor(obs[0]).unsqueeze(0)
                probs = self.agents[0].actor(obs_t)
                dist = torch.distributions.Categorical(probs)
                episode_entropy.append(dist.entropy().item())
                
                # Execute actions
                next_obs, env_rewards, done, info = self.env.step(actions)
                
                # Get LLM rewards
                llm_rewards = llm_feedback(
                    state=obs,
                    actions=actions,
                    agent_positions=self.env.agent_positions,
                    goal_positions=self.env.goals,
                    obstacles=self.env.obstacles
                )
                
                # Combined rewards
                combined_rewards = [
                    env_rewards[i] + 0.5 * llm_rewards[i]
                    for i in range(len(self.agents))
                ]
                
                episode_reward += sum(combined_rewards)
                episode_collisions = info["collisions"]
                positions_history.append(list(self.env.agent_positions))
                
                obs = next_obs
                step_count += 1
            
            # Store episode results
            self.results["rewards"].append(episode_reward)
            self.results["lengths"].append(step_count)
            self.results["successes"].append(info["success"])
            self.results["collisions"].append(episode_collisions)
            self.results["entropy"].append(np.mean(episode_entropy))
            self.results["value_estimates"].append(np.mean(episode_values))
            self.results["agent_positions_history"].append(positions_history)
            
            if (ep + 1) % 10 == 0:
                print(f"  Completed {ep + 1}/{self.num_eval_episodes} episodes")
        
        print("Evaluation complete!")
    
    def compute_statistics(self):
        """Compute comprehensive evaluation statistics."""
        stats = {
            "environment": self.env_name,
            "episodes_evaluated": self.num_eval_episodes,
            
            # Reward statistics
            "mean_reward": np.mean(self.results["rewards"]),
            "std_reward": np.std(self.results["rewards"]),
            "min_reward": np.min(self.results["rewards"]),
            "max_reward": np.max(self.results["rewards"]),
            
            # Success statistics
            "success_rate": np.mean(self.results["successes"]),
            "total_successes": np.sum(self.results["successes"]),
            
            # Episode length statistics
            "mean_length": np.mean(self.results["lengths"]),
            "std_length": np.std(self.results["lengths"]),
            "timeout_rate": np.mean([l >= 200 for l in self.results["lengths"]]),
            
            # Collision statistics
            "mean_collisions": np.mean(self.results["collisions"]),
            "total_collisions": np.sum(self.results["collisions"]),
            "collision_free_episodes": np.sum([c == 0 for c in self.results["collisions"]]),
            
            # Policy statistics
            "mean_entropy": np.mean(self.results["entropy"]),
            "mean_value_estimate": np.mean(self.results["value_estimates"]),
            
            # Efficiency metrics
            "reward_per_step": np.sum(self.results["rewards"]) / np.sum(self.results["lengths"]),
            "success_per_collision": np.sum(self.results["successes"]) / (np.sum(self.results["collisions"]) + 1)
        }
        
        return stats
    
    def print_report(self):

        stats = self.compute_statistics()
        
        print("\n" + "="*60)
        print(f"EVALUATION REPORT: {self.env_name} Environment")
        print("="*60)
        
        print("\n📊 OVERALL PERFORMANCE:")

        print(f"  Timeout Rate:       {stats['timeout_rate']:.1%}")
        print(f"  Collision Rate:     {stats['mean_collisions']:.2f} per episode")
        
        print("\n💰 REWARD ANALYSIS:")
        print(f"  Mean Reward:        {stats['mean_reward']:.2f} ± {stats['std_reward']:.2f}")
        print(f"  Reward Range:       [{stats['min_reward']:.2f}, {stats['max_reward']:.2f}]")

# inference.py - Run trained agent on new scenarios

import torch
import numpy as np
from agent import MAPPOAgent
from env.simple import SimpleEnv
from env.medium import MediumEnv
from env.complex import ComplexEnv
from env.dynamic import DynamicEnv


class RLInference:
    """
    Run inference with trained RL agents.
    """
    
    def __init__(self, env_name="Simple", model_path=None):
        self.env_name = env_name
        self.model_path = model_path
        self.env = None
        self.agents = []
        
    def load_environment(self):
        """Load environment."""
        ENV_MAP = {
            "Simple": SimpleEnv,
            "Medium": MediumEnv,
            "Complex": ComplexEnv,
            "Dynamic": DynamicEnv
        }
        self.env = ENV_MAP[self.env_name]()
        
    def initialize_agents(self, trained_weights=None):
        """Initialize agents with optional trained weights."""
        obs = self.env.reset()
        obs_dim = len(obs[0])
        
        self.agents = [MAPPOAgent(obs_dim=obs_dim, action_dim=5) 
                       for _ in range(self.env.num_agents)]
        
        # Load trained weights if provided
        if trained_weights and self.model_path:
            for i, agent in enumerate(self.agents):
                checkpoint = torch.load(f"{self.model_path}/agent_{i}.pt")
                agent.actor.load_state_dict(checkpoint['actor'])
                agent.critic.load_state_dict(checkpoint['critic'])
                # Set to evaluation mode
                agent.actor.eval()
                agent.critic.eval()
    
    def predict(self, deterministic=False):
        """
        Run one episode with trained agents.
        
        Args:
            deterministic: If True, always choose best action (no sampling)
        
        Returns:
            Dictionary with episode results
        """
        obs = self.env.reset()
        episode_data = {
            "positions": [list(self.env.agent_positions)],
            "actions": [],
            "rewards": [],
            "total_reward": 0.0,
            "steps": 0,
            "success": False,
            "collisions": 0
        }
        
        done = False
        
        while not done:
            actions = []
            
            for i, agent in enumerate(self.agents):
                obs_t = torch.FloatTensor(obs[i]).unsqueeze(0)
                
                with torch.no_grad():
                    probs = agent.actor(obs_t)
                    
                    if deterministic:
                        # Take most probable action
                        action = torch.argmax(probs, dim=-1).item()
                    else:
                        # Sample from distribution
                        dist = torch.distributions.Categorical(probs)
                        action = dist.sample().item()
                
                actions.append(action)
            
            # Execute actions
            next_obs, rewards, done, info = self.env.step(actions)
            
            # Store data
            episode_data["actions"].append(actions)
            episode_data["rewards"].append(sum(rewards))
            episode_data["positions"].append(list(self.env.agent_positions))
            episode_data["total_reward"] += sum(rewards)
            episode_data["steps"] += 1
            episode_data["collisions"] = info["collisions"]
            
            obs = next_obs
        
        episode_data["success"] = info["success"]
        
        return episode_data
    
    def run_multiple_episodes(self, num_episodes=10, deterministic=False):
        """Run multiple inference episodes."""
        results = []
        
        for ep in range(num_episodes):
            episode_result = self.predict(deterministic=deterministic)
            results.append(episode_result)
            
            print(f"Episode {ep+1}/{num_episodes}: "
                  f"Reward={episode_result['total_reward']:.2f}, "
                  f"Steps={episode_result['steps']}, "
                  f"Success={episode_result['success']}, "
                  f"Collisions={episode_result['collisions']}")
        
        # Aggregate statistics
        total_rewards = [r['total_reward'] for r in results]
        successes = [r['success'] for r in results]
        collisions = [r['collisions'] for r in results]
        steps = [r['steps'] for r in results]
        
        summary = {
            "episodes": num_episodes,
            "mean_reward": np.mean(total_rewards),
            "success_rate": np.mean(successes),
            "mean_collisions": np.mean(collisions),
            "mean_steps": np.mean(steps),
            "deterministic": deterministic
        }
        
        print(f"\nInference Summary ({num_episodes} episodes):")
        print(f"  Mean Reward: {summary['mean_reward']:.2f}")
        print(f"  Success Rate: {summary['success_rate']:.1%}")
        print(f"  Mean Collisions: {summary['mean_collisions']:.2f}")
        print(f"  Mean Steps: {summary['mean_steps']:.1f}")
        
        return results, summary


# Example usage
if __name__ == "__main__":
    # Create inference engine
    inference = RLInference(env_name="Medium")
    
    # Load environment
    inference.load_environment()
    
    # Initialize agents (without trained weights for demo)
    inference.initialize_agents(trained_weights=False)
    
    # Run inference
    print("Running inference with trained agents...")
    results, summary = inference.run_multiple_episodes(
        num_episodes=10, 
        deterministic=True  # Use best actions
    )
